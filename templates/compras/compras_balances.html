Filtros de búsqueda

¿Cómo usar los filtros?
Este módulo permite analizar las compras realizadas por la empresa.

  • Cuenta bancaria: Filtra compras según la cuenta utilizada para el pago.
  • Sucursal: Muestra compras asociadas a una sucursal específica.
  • Productor: Permite ver compras realizadas a un productor determinado.
  • Producto: Selecciona el producto y variedad para analizar sus compras.
  • Tipo de pago: Filtra por método de pago (efectivo, transferencia, cheque, etc).
  • Año y periodo: Ajusta el año y el periodo (diario, semanal, mensual) para ver compras en ese rango.
  • Meses: Si el periodo es mensual, puedes seleccionar varios meses para comparar.
  • Fechas: Para análisis diario, selecciona un día específico o un rango de fechas.

Haz clic en Aplicar Filtros para actualizar la información. Usa Restablecer para limpiar todos los filtros y ver el acumulado general de compras.
