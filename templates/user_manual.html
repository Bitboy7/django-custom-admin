{% trans "Registrar Rendimientos" %}

{% trans "Para inversiones tipo SALIDA, puedes registrar los rendimientos obtenidos a lo largo del tiempo:" %}

  1. {% trans "Accede a la inversión en el panel de administración." %}
  2. {% trans "Desplázate a la sección 'Rendimientos' dentro del formulario." %}
  3. {% trans "Haz clic en 'Agregar rendimiento'." %}
  4. {% trans "Registra la fecha y monto del rendimiento." %}
  5. {% trans "Opcionalmente, especifica el tipo de rendimiento (Dividendos, Intereses, Ganancias de capital, etc.)." %}
  6. {% trans "El sistema calculará automáticamente el porcentaje de rendimiento." %}

{% trans "Cálculo del ROI (Retorno sobre Inversión)" %}

{% trans "Fórmula de Cálculo" %}

ROI (%) = (Total Rendimientos ÷ Inversión Inicial) × 100

{% trans "Ejemplo:" %}

  • {% trans "Inversión inicial:" %} $10,000
  • {% trans "Rendimientos acumulados:" %} $1,500
  • {% trans "ROI:" %} (1,500 ÷ 10,000) × 100 = 15%

{% trans "Puntos Importantes sobre el ROI" %}

  • {% trans "Solo aplica a inversiones tipo SALIDA:" %} {% trans "El ROI únicamente se calcula para inversiones donde el dinero sale de la empresa." %}
  • {% trans "Rendimientos acumulados:" %} {% trans "El sistema suma todos los rendimientos registrados para calcular el ROI total." %}
  • {% trans "Múltiples registros:" %} {% trans "Puedes registrar varios rendimientos para una misma inversión a lo largo del tiempo." %}
  • {% trans "Moneda consistente:" %} {% trans "Todos los cálculos se realizan en la misma moneda configurada." %}
  • {% trans "ROI individual vs promedio:" %} {% trans "Cada inversión tiene su ROI individual. El dashboard muestra el ROI promedio ponderado del portafolio completo." %}

{% trans "Interpretación de Resultados" %}

  • {% trans "ROI Positivo:" %} {% trans "La inversión está generando ganancias. Mientras mayor el porcentaje, mejor el rendimiento." %}
  • {% trans "ROI = 0%:" %} {% trans "No se han registrado rendimientos aún o la inversión está en punto de equilibrio." %}
  • {% trans "ROI Negativo:" %} {% trans "La inversión ha generado pérdidas. Requiere análisis y posible ajuste de estrategia." %}
